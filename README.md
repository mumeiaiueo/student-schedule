# 学生予定管理 PWA シンプル版

## コンセプト
ごちゃごちゃしない学生向け予定管理アプリです。

## 画面
- ホーム: 今日の予定 + 締切近い課題
- 追加: タイトル、種類、日付、時刻、通知、メモ
- カレンダー: 小さめ月表示
- 時間割: 別ページ
- 設定: 通知、バックアップ、復元、全削除

## 追加時の演出
- 「追加しました！」のトースト表示
- ボタンが一瞬「追加しました！」に変化
- タイトル欄とメモ欄を白紙化
- タイトル欄に自動フォーカス
- 種類・日付・時刻・通知は前回のまま保持

## 注意
この試作版は端末内保存です。強力なバックグラウンド通知にはサーバー実装が必要です。


## 通知モード更新
- しつこめ: 3日前、2日前、前日、12時間前、9時間前、6時間前、3時間前、1時間前、30分前、締切/開始時刻。課題は未完了なら締切後30分、1時間、3時間にも通知。
- 通知あり: 1時間前、30分前、10分前、締切/開始時刻。
- 通知なし: 通知しない。


## 朝のまとめ通知
- デフォルトは8:00
- 設定からオン/オフと時刻変更が可能
- 今日の予定件数と、近い課題をまとめて通知
- 同じ日は1回だけ通知
